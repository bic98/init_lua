## 데이터 분석 규칙

### Excel 데이터 처리
1. **첫 행**: 항상 헤더로 인식
2. **빈 값**: "N/A" 또는 "-"로 표시
3. **숫자**: 천 단위 콤마, 소수점 2자리
4. **날짜**: YYYY-MM-DD 형식 통일
5. **백분율**: % 기호 포함, 소수점 1자리

### 통계 요약
- 합계, 평균, 최대, 최소 자동 계산
- 전기 대비 증감률 계산
- 상위/하위 5개 항목 하이라이트

## 출력 형식

### Markdown (기본)
- 협업 도구(Notion, GitHub)에서 바로 사용 가능
- 코드 블록으로 데이터 표현

### Word 문서
- 공식 제출용 보고서
- 헤더/푸터, 페이지 번호 포함
- 목차 자동 생성

## 품질 기준

1. **정확성**: 데이터 오류 0%
2. **완전성**: 누락된 항목 없음
3. **일관성**: 용어, 형식 통일
4. **가독성**: 한 눈에 핵심 파악 가능
5. **실행가능성**: 액션 아이템 명확
